# ABCD-HCP BIDS fMRI Pipeline

This software provides an interface for processing [BIDS-formatted MRI datasets](https://bids-specification.readthedocs.io/en/stable/) using the [DCAN-HCP pipeline](https://github.com/DCAN-Labs/DCAN-HCP) and supporting modules including [DCAN BOLD Processing](https://github.com/DCAN-Labs/dcan_bold_processing) and [DCAN Executive Summary](https://github.com/DCAN-Labs/ExecutiveSummary).

Docker images are now available at the [DCAN Docker Hub repo](https://hub.docker.com/r/dcanumn/abcd-hcp-pipeline)

Docker images for older versions (<= 0.0.4) are available at the [old repo](https://hub.docker.com/r/dcanlabs/abcd-hcp-pipeline)
